Privacy Policy

Effective Date: February 1, 2026 • Version 1.0

GamerHub ("we", "our", or "us") provides this Privacy Policy to describe our practices regarding the collection, use, and disclosure of information through the GamerHub Chrome Extension (the "Service"). By using the Service, you agree to the collection and use of information in accordance with this policy.

1. Data Collection and Storage

We adhere to a strict local-first architecture. GamerHub does not create user accounts on external servers, nor do we track your browsing history outside of the specific game services required for functionality.

  • Local Storage: All user preferences, friend lists, and API keys are stored locally on your device using the browser's chrome.storage.local API.
  • No Remote Database: We do not operate a backend server to store personal user data. We have no access to your data.

2. Permissions and Technical Usage

The extension requests the minimum set of permissions necessary to operate. Below is a transparent breakdown of each permission:

Cookies

Used strictly to authenticate your requests to steamcommunity.com. The extension uses your existing browser session cookies to fetch your friend list and profile status securely. We do not access, store, or transmit your credentials (username/password).

Host Permissions

The extension makes direct XMLHttpRequests (XHR) or Fetch API calls to the following third-party endpoints to retrieve real-time data:

  • Valve Corporation: api.steampowered.com (Friend activity)
  • Faceit: open.faceit.com (Player statistics)
  • CheapShark & GamerPower: cheapshark.com (Price data)
  • Cloudflare & AWS: Used exclusively for the network latency and speed test features.

3. Monetization and Affiliate Disclosure

GamerHub is provided free of charge. To sustain development costs, the Service may include affiliate links within the "Deals" section.

This means if you click on a link to a third-party retailer (such as Instant Gaming, G2A, or CDKeys) and make a purchase, we may receive a commission. This process is anonymous and does not increase the price you pay.

4. Network Performance Tools

The "Network" tab includes a ping and speed test utility. Using this feature initiates data transfer between your browser and public test servers (Cloudflare/AWS) to measure latency. This activity is transient and no logs of your IP address are retained by us.

5. Data Retention and Deletion

Since data is stored locally:

  • Retention: Data persists as long as the extension is installed.
  • Deletion: You may delete all stored data by uninstalling the extension from your browser. Specific API keys can be removed via the in-app settings menu.
